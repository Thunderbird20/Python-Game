# Assets and Attribution

Place game images, audio, and other large files in the `assets/` directory.
Keep binary data out of source control if files are large; consider using
Git LFS for big media.

Attribution

If you include third-party assets, add an entry here with:

- Asset name
- Source / author
- License (e.g., CC-BY, CC0)
