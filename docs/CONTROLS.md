# Controls

Default controls used by the game. Adjust per platform and input mode.

Keyboard (PC)
- Arrow keys / WASD — movement
- `Space` — action / jump / confirm
- `Esc` — pause / open menu

Gamepad
- Left stick / D-pad — movement
- A / Bottom button — action / confirm
- Start / Menu button — pause

Custom mapping

If the game supports key remapping, edit the `config/controls.json` or use
the in-game options menu to change bindings.
